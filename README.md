# NorthStar Urban Mobility and Logistics — Database and Analytics

## Module
Databases and Analytics (CP60056E)

## Overview
This repository contains the analytical work for the NorthStar Urban Mobility
and Logistics case study. The project investigates operational inefficiencies,
service reliability failures, and data fragmentation issues using SQL in R,
R analytics, Python data processing, and MongoDB.

## Repository Structure
NorthStar_database/
├── data/

│   └── cleaned/        # Cleaned dataset files after preprocessing
